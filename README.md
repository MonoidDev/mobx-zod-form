# mobx-zod-form

Data-first form builder based on [MobX](https://mobx.js.org/) & [Zod](https://zod.dev/).

## Supported types

1. `ZodString`
2. `ZodNumber`
3. `ZodBoolean`
4. `ZodEnum`
5. `ZodOptional`
6. `ZodNullable`
7. `ZodLiteral`
8. `ZodAny`
9. `ZodObject`
10. `ZodArray`
11. `ZodDefault`
12. `ZodDiscriminatedUnion`
13. `ZodEffects`
14. `MobxZodBox`
