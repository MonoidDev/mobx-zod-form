# mobx-zod-form

Data-first form builder based on [MobX](https://mobx.js.org/) & [Zod](https://zod.dev/).

## Supported types

1. `ZodString`
2. `ZodNumber`
3. `ZodBoolean`
4. `ZodEnum`
5. `ZodLiteral`
6. `ZodObject`
7. `ZodArray`
8. `ZodDefault`
9. `z.discriminatedUnion`
10. `z.optional`
11.  `z.nullable`
 
12. Custom atomic types.
