# mobx-zod-form

Data-first form builder based on [MobX](https://mobx.js.org/) & [Zod](https://zod.dev/).

## Supported types

1. `ZodString`

2. `ZodNumber`

3. `ZodBoolean`

4. `ZodObject`

5. `ZodArray`

6. `ZodDefault`

7. `z.discriminatedUnion`
   
8. `z.optional`

9. `z.nullable`
 
10. Custom atomic types.
